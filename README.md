# 🚀 Dokanplus - Modern Arabic Landing Page

### 🔗 [ Live Demo ](https://dokan-plus-website.vercel.app/) 🌐

A high-performance, fully responsive landing page designed for the **Dokanplus** platform. This project showcases advanced Frontend techniques, focusing on Arabic typography (RTL) and interactive user experiences.

## 🛠️ Tech Stack & Libraries

* **🎨 HTML5 & CSS3**: Semantic structure with custom styling.
* **📱 Bootstrap**: For a mobile-first responsive grid system.
* **⚡ JavaScript (ES6)**: Custom interactions and UI logic.
* **🔍 Isotope.js**: Advanced layout and filtering for product/feature sections.
* **✨ AOS (Animate On Scroll)**: Smooth entrance animations for a better UX.
* **🖋️ Google Fonts**: Integrated "Tajawal" for professional Arabic typography.

---

## ✨ Key Features

* **🌍 Full RTL Support**: Optimized specifically for Arabic-speaking users with right-to-left layout consistency.
* **🖱️ Interactive UI**: Includes hover effects, animated transitions, and dynamic filtering.
* **🔝 Modern Hero Section**: High-impact introduction with clear Call-to-Action (CTA) buttons.
* **📲 Responsive Design**: Seamlessly adapted for Desktops, Tablets, and Mobile devices.
* **💳 Dynamic Pricing**: Clear presentation of service plans and platform benefits.

---

## 📂 Project Structure

* **`📄 index.html`**: Main landing page structure.
* **`🎨 style.css`**: Custom styles including RTL logic and typography.
* **`⚙️ main.js`**: Main logic for interactions and UI controls.
* **`📦 isotope.pkgd.js`**: Library for dynamic item filtering.
* **`🖼️ assets/Images/`**: All visual assets, icons, and logos.
* **`🔠 assets/Fonts/Tajawal`**: Professional Arabic font families.
